# Roadmap

最后更新：2026-03-30

## 产品目标

把 `hxz-fortune-telling` 打磨成一个普通用户可直接使用、结果易懂、输入成本可控的运势分析产品。

## 当前阶段

当前阶段：`v1.x`

当前重点：先把输入、分析、结果表达和时间段建议跑顺，再逐步提升结果表达质量和可复用能力。

## 版本规划

### v1.1
目标：完成从输入到结果的最短闭环。
状态：进行中
重点：
- 标准/大概两种输入模式
- 时间段建议与分段输出
- 缺失时辰的精度提示

### v1.2
目标：让结果更像建议而不是报告。
状态：未开始
重点：
- 先结论后依据
- 简明模式 / 详细模式
- 风险与免责声明表达优化

### v1.3
目标：提升输入精度与结果差异提示能力。
状态：未开始
重点：
- 增加出生地
- 进阶模式
- 标准与进阶结果差异提示

### v1.4
目标：增强复盘与复用能力。
状态：未开始
重点：
- 历史记录增强
- 结果对比
- 快捷时间段入口

### v1.5
目标：统一全链路文案和可信度表达。
状态：未开始
重点：
- 文案统一
- 可信度提示
- 新手引导

## 暂不纳入当前阶段

- 专业咨询师工作台
- PDF 报告导出
- 复杂账户体系
- 专业级多流派引擎
