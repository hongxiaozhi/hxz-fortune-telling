# Release Plan: v1.1

最后更新：2026-03-31  
状态：已完成

## 版本目标

完成从用户输入到分析结果返回的最短闭环，并保证结果结构可读、可解释、可继续扩展。

## 用户可见变化

- 用户可以直接输入出生信息并获得分析结果
- 可以选择未来时间段并得到分段建议
- 缺失出生时辰时仍可获得基础结论，并看到精度提示
- 结果页按“免责说明 -> 一句话结论 -> 八字简表 -> 五行占比 -> 整体建议 -> 分段建议”的顺序展示

## 范围边界

本版本包含：

- 基础输入表单与校验
- 输入页、加载态、结果页、历史抽屉四类页面结构
- 结果页主结构与基础表达
- 结果文案中文化与页面语言一致性修复
- 时间段分段建议
- 精度说明与升级提示

本版本不包含：

- 出生地进阶模式
- 结果对比
- 多设备同步

延期到后续版本：

- 出生地与进阶模式 -> `v1.3`
- 历史与结果对比 -> `v1.4`
