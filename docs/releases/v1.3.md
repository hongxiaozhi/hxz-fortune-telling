# Release Plan: v1.3

最后更新：2026-03-31  
状态：进行中

## 版本目标

提升输入精度与结果差异提示能力，在保持当前主链路结构稳定的前提下，让用户更清楚哪些输入会影响结果细节。

## 当前版本承接关系

- `v1.2` 已完成结果页表达顺序、阅读模式和风险提示优化
- `v1.3` 开始进入输入精度增强阶段，重点补齐出生地和标准 / 进阶模式差异提示
- 当前继续沿用根目录统一 `docker compose` 运行方式

## 范围边界

本版本包含：

- 新增出生地输入字段
- 新增标准 / 进阶分析模式
- 在结果中补充“当前使用了哪些输入条件”的差异提示
- 根据标准 / 进阶模式生成轻量结果差异
- 保持现有 `POST /api/fortune/analyze` 主体结构可被前端直接消费

本版本不包含：

- 历史结果对比
- 多设备同步
- 专业咨询师工作台

延期到后续版本：

- 历史与结果对比 -> `v1.4`
- 新手引导与全链路文案统一 -> `v1.5`
