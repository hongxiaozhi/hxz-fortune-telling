# Release Plan: v1.5

最后更新：2026-03-31  
状态：进行中

## 版本目标

统一全链路文案和可信度表达，让新手第一次进入页面时就知道怎么填、怎么看、该把结果当成什么来用。

## 当前版本承接关系

- `v1.4` 已完成历史记录增强、结果对比、输入回填与本地 Docker 验证
- `v1.5` 重点承接文案统一、可信度提示、新手引导
- 当前继续沿用根目录统一 `docker compose` 运行方式

## 范围边界

本版本包含：

- 输入页新手引导
- 结果页可信度提示
- 页面文案统一与语气收口
- 保持现有 `POST /api/fortune/analyze` 顶层结构稳定

本版本不包含：

- 新算法引擎
- 多设备同步
- 账户体系
- 专业咨询师工作台

延期到后续版本：

- PDF 报告导出 -> 后续版本
- 更复杂的多次结果结构化归纳 -> 后续版本
