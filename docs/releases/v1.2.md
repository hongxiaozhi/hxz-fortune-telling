# Release Plan: v1.2

最后更新：2026-03-31  
状态：进行中

## 版本目标

让结果更像建议而不是报告，在保持现有接口结构稳定的前提下，优化结果表达顺序与阅读体验。

## 当前版本承接关系

- `v1.1` 已完成主链路、中文结果回归和线上路由验证
- `v1.2` 不重做输入与接口结构，重点优化结果呈现方式
- 当前仍沿用根目录统一 `docker compose` 运行方式

## 范围边界

本版本包含：

- 结果页“先结论后依据”的进一步强化
- 简明模式 / 详细模式
- 风险与免责表达优化
- 不改变现有 `POST /api/fortune/analyze` 返回结构的前提下优化阅读体验

本版本不包含：

- 出生地
- 进阶模式
- 历史对比
- 多设备同步

延期到后续版本：

- 出生地与进阶模式 -> `v1.3`
- 历史与结果对比 -> `v1.4`
