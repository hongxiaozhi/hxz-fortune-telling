# Changelog

## v1.1.0 - 2026-03-30

### Added

- 健康检查接口
- 运势分析接口基础校验
- 7 天分段建议结果结构

### Changed

- 结果输出统一为可直接消费的 JSON 结构
- 缺失出生时辰时增加精度降级与升级提示
- 前端结果页改为“先结论后细节”的信息顺序
- 输入页、加载态、结果页、历史抽屉统一为稳定的 `v1.1` 主链路
- 补充 Windows 本地 Docker 构建与根目录统一编排验证记录

### Fixed

- 修复前端页面文案乱码和损坏模板导致的主流程不可用问题
- 修复后端文件损坏导致服务无法启动的问题
- 修复线上 Docker 与 Nginx 路由不一致的问题

### Notes

- 当前部署方式已统一为根目录 Docker 编排 + Nginx 反向代理
- 已确认 Windows 本地可通过 `docker compose up -d --build hxz-fortune` 启动并访问修改后的页面版本
