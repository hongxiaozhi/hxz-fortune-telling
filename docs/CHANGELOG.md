# Changelog

## v1.1.1 - 2026-03-31

### Changed

- 分析结果中的免责说明、精度说明、升级提示统一改为中文
- 八字简表示例值改为中文干支
- 整体建议和分段建议改为中文表达
- 健康检查版本号升级为 `v1.1.1`

### Fixed

- 修复结果页虽然是中文界面，但接口返回仍为英文内容的问题

### Notes

- 本版本为 `v1.1` 的补丁版本，保持结果结构不变，只修正文案语言与版本记录
- 已在 Windows 本地通过 `docker compose up -d --build hxz-fortune` 完成重建，并确认 `v1.1.1` 返回中文结果

## v1.1.0 - 2026-03-30

### Added

- 健康检查接口
- 运势分析接口基础校验
- 7 天分段建议结果结构

### Changed

- 结果输出统一为可直接消费的 JSON 结构
- 缺失出生时辰时增加精度降级与升级提示
- 前端结果页改为“先结论后细节”的信息顺序
- 输入页、加载态、结果页、历史抽屉统一为稳定的 `v1.1` 主链路
- 补充 Windows 本地 Docker 构建与根目录统一编排验证记录

### Fixed

- 修复前端页面文案乱码和损坏模板导致的主流程不可用问题
- 修复后端文件损坏导致服务无法启动的问题
- 修复线上 Docker 与 Nginx 路由不一致的问题

### Notes

- 当前部署方式已统一为根目录 Docker 编排 + Nginx 反向代理
- 已确认 Windows 本地可通过 `docker compose up -d --build hxz-fortune` 启动并访问修改后的页面版本
