# Changelog

## v1.4.0 - 2026-03-31

### Added

- 历史记录摘要信息增强，展示模式与分析区间
- 基于历史记录的轻量结果对比
- 输入页快捷时间段入口
- 历史记录支持删除单条记录与清空全部记录
- 结果页新增“本次差异总结”，帮助解释当前结果与对比记录为什么不同
- 结果页新增历史记录概览，展示最近记录数量与模式分布
- `docs/releases/v1.4.md`

### Changed

- 健康检查版本号升级为 `v1.4.0`
- 项目当前阶段从 `v1.3` 切换到 `v1.4`
- `v1.4` 从第一批基础能力推进到第二批复盘与管理增强

### Notes

- 本次继续保持分析接口结构稳定，重点增强前端复盘与复用能力
- 已完成本地 `docker compose up -d --build hxz-fortune` 构建与运行验证

## v1.3.1 - 2026-03-31

- 健康检查版本号升级为 `v1.3.1`

### Notes

- 本次仍保持原有 API 顶层结构不变，只增强差异提示能力
